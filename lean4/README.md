## Architecture: Two-Theorem Structure

### Theorem I: Closure Theorem (Conditional Gronwall)

**Statement.** Given:
- `[Geom]` Triad support geometry with offset C0 = 2
- `[Alg]` Finite Fourier Bernstein inequality (Cauchy-Schwarz on mode set)
- `[Alg]` Low-shell H1 energy bound
- `[Def]` Shell decomposition P_k = P_loc + P_tail
- `[Def]` Dissipation identity D_k = nu * k^2 * Xi_k
- `[Alg]` hcoeff_l2_to_localH1 (shell-block coefficient bound)

Then there exists N0 such that for all m:

    E(N0 + m) <= (1 + M*C)^m * E(N0)

**Discharge chain:**
```
NSUnitWidthEuclideanRadius  [Geom] actual Euclidean radius on Mode
  -> NSUnitWidthTriadGeometry  [Geom] +2 shell-index triangle estimate
    -> NSLowShellLocalClosureActual  [Geom] tail vanishing on low shells
      -> NSFiniteBandEnergy  [Alg] low-shell energy bounds
        -> NSFiniteFourierBernstein  [Alg] Cauchy-Schwarz on finite mode set
          -> NSLocalH1EnergyBound  [Alg] H1 <= Ncut * sqrt(E_<N)
            -> NSLocalPairingToDissipation  [Alg] pairing/dissipation ratio
              -> NSFiniteSourceConditionalGronwall  [Alg] discrete Gronwall
```

### Theorem II: NS Realization Theorem

**Statement.** Given true Navier-Stokes fields satisfying `[PDE]` assumptions,
the hypotheses of Theorem I are discharged.

**Discharge chain:**
```
NSGalerkinExistenceActual  [ODE] concrete Fin m -> R state space
  -> NSGalerkinCoeffODEProductRuleProof  [ODE] product rule for Xi
    -> NSGalerkinCoeffProductRuleActual  [ODE] FTC integration
      -> NSGalerkinIntervalIntegralActual  [ODE] interval averages
        -> NSGalerkinShellStepIdentity  [Alg] shellwise step identity
          -> NSGalerkinEnstrophyIdentity  [Alg] total enstrophy identity
            -> NSGalerkinHinc  [Alg] integrated hinc
              -> NSIntegratedHinc  [Alg] scaled budget trajectory
```

### Theorem III: No-Blowup Master Theorem

**Statement.** Given the discrete Gronwall envelope and bootstrap regeneration,
no minimal blow-up scenario can occur.

**Discharge chain:**
```
NSStrainSupBootstrap  [Alg] bootstrap strain regeneration (low/high decomposition)
  -> NSContinuationCriterion  [Alg] inductive continuation criterion
    -> NSNoBlowupMaster  [Alg] contradiction to minimal blow-up
```

are all K_max-independent, enabling Arzela-Ascoli extraction.
